# Archive Directory

This directory contains old test files and temporary artifacts that are no longer needed for the main project but are kept for reference.

## Contents

- **test_*.csv**: Temporary test files used during development
- **single_test.csv**: Single project test file
- **test_subset.csv**: Small subset of projects for testing
- **test_downloader.py**: Test script for the downloader module
- **sample_projects.csv**: Original sample projects file

## Note

These files can be safely deleted if no longer needed. They were used during development and testing phases.
